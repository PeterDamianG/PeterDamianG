### What Is It? - ¿Qué es esto? <a name="what"></a>

An <b>unofficial</b> Template for <b>NextJS.</b> It's a help with start your proyect with <b>ESlint, Prettier, Next-Themes, Jest + TestingLibrary, Cypress</b> and more features now/incoming.

Una plantilla <b>no oficial,</b> para usar <b>NextJS.</b> Es una ayuda para comenzar tus proyectos con <b>ESLint, Prettier, Next-Themes, Jest + TestingLibrary, Cypress</b> y más caracteristicas ahora/proximamente.

## Create a new App - Crear una nueva App <a name="create"></a>

##### We have this ways - Tenemos las siguientes maneras:

1. In your favorite console - En tu consola favorita:

`git clone https://github.com/PeterDamianG/template-next.git`

`cd template-next`

`npm install`

And done, that command is to run <b>next</b> in mode <b>develoment</b> - Y listo, el siguiente comando es para correr <b>next</b> en modo <b>desarrollo:</b>

`npm run dev`

Go here to see - Ve aquí para ver el resultado:

[LocalHost](http://localhost:3000)

2. Download with your favorite browser - O descargandolo desde tu navegador:

[Download-Zip](https://github.com/PeterDamianG/template-next/archive/main.zip)

UnZip in directory and in - Descomprimir en carpeta o directorio y dentro:

`npm install`

And done, that command to run <b>next</b> in mode <b>develoment</b> - Y listo, el siguiente comando es para correr <b>next</b> en modo <b>desarrollo:</b>

`npm run dev`

Go here to see - Ve aquí para ver el resultado:

[LocalHost](http://localhost:3000)

3. Generate a new repository from this template in your <b>GitHub</b> - Generar un nuevo repositorio desde esta plantilla en tu <b>GitHub:</b>

[Click to Generate](https://github.com/PeterDamianG/template-next/generate)

## Available Scripts - Scripts Disponibles <a name="scripts"></a>

### npm run export

Run a script to build with <b>next</b> and export all proyect <b>if possible,</b> to static files in the folder `/.out`.
Ready to <b>deploy</b> or see in your browser.

Corre un script para construir con <b>next</b> y exportar todo el proyecto <b>si es posible,</b> como archivos estáticos en la carpeta `/.out`.
Listo para <b>desplegar</b> o ver en tu navegador.

### npm run see-export

A shortcut to open <b>result of export.</b>

Un atajo para abrir el <b>resultado</b> de usar el comando de export.

### npm run dev

Runs the <b>app</b> in the <b>development mode.</b>
Open http://localhost:3000 to view it in the browser.
The page will reload if you make edits.
You will also see any lint errors in the console.

Inicia la <b>aplicacion</b> en <b>modo de desarrollo.</b>
Abre http://localhost:8080 para verlo en el navegador.
La página se recarga sí haces cambios.
También veras cualquier error en la consola.

### npm run dev-debug

Runs the <b>app</b> in the <b>development mode to debug.</b>
Open http://localhost:3000 to view it in the browser.
The page will reload if you make edits.
You will also see any lint errors in the console.

Inicia la <b>aplicacion</b> en <b>modo de desarrollo para debuggear.</b>
Abre http://localhost:8080 para verlo en el navegador.
La página se recarga sí haces cambios.
También veras cualquier error en la consola.

More info - Más info: [Next Debug](https://nextjs.org/docs/advanced-features/debugging)

### npm run build

Builds run a internal commando of <b>next</b>, result to the `/.next` folder.

Construye es un comando interno de <b>next</b>, el resultado en la carpeta `/.next`.

### npm run start or npm start

Runs the <b>app</b> in the <b>production mode.</b>
Open http://localhost:3000 to view it in the browser.

Corre la <b>aplicación</b> en <b>modo de producción.</b>
Abre http://localhost:8080 para verlo en el navegador.

### npm run prepare

It's needed for run <b>husky. Shouldn't be necessary run.</b>

Es necesario para ejecutar <b>husky. No debería ser necesario usarlo.</b>

### npm run prettier

Use <b>Prettier</b> to check/write files on the `/src` folder.

Usa <b>Prettier</b> para revisar/escribir los archivos en la carpeta `/src`.

### npm run eslint

Use <b>ESlint</b> to check/write files on the `/src` folder.

Usa <b>ESlint</b> para revisar/escribir los archivos en la carpeta `/src`.

### npm run test

Launches the application test runner <b>Jest</b>, find test on the `/src` folder.

Lanza el corredor de pruebas de <b>Jest</b>, que busca los test en la carpeta `/src`.

### npm run coverage

Run all test and build a <b>coverage</b> folder in `/coverage`.

Corre todas las pruebas y construye una carpeta de <b>coverturas</b> en `/coverage`.

### npm run see-coverage

A shortcut to open result of <b>coverage.</b>

Un atajo para abrir el resultado de usar el comando de <b>coverage.</b>

### npm run docs

Run tool <b>JSDOC</b> for generate documentation about the proyect, folder in `/docs`.

Corre la herramienta <b>JSDOC</b> para generar documentación sobre el proyecto, la carpeta será `/docs`.

### npm run see-docs

A shortcut to open result of <b>docs.</b>

Un atajo para abrir el resultado de usar el comando de <b>docs.</b>

### npm run cypress

A shortcut to open <b>Cypress.</b>

Un atajo para abrir <b>Cypress.</b>

### npm run cypress-headless

A shortcut to open <b>Cypress</b> headless mode.

Un atajo para abrir <b>Cypress</b> en modo sin cabezera.
